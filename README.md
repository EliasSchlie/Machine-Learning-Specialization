# Machine Learning Specialization — Solutions

Solutions to the programming assignments from the [Machine Learning Specialization](https://www.coursera.org/specializations/machine-learning-introduction) by Stanford University & DeepLearning.AI (Andrew Ng).

## Courses

1. **Supervised Machine Learning** — linear regression, logistic regression, regularization
2. **Advanced Learning Algorithms** — neural networks, decision trees, practical ML advice
3. **Unsupervised Learning** — clustering, anomaly detection, recommender systems, reinforcement learning

Each course folder is organized by week with Jupyter notebooks.
